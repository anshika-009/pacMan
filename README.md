# Java Pac-Man Desktop Game

A classic, retro-inspired 2D Pac-Man desktop application built from scratch in Java using the Swing framework. Navigate through the maze, gobble up all the pellets, and avoid the colorful ghosts to achieve the highest score!

## 🚀 Features

*   **Classic Gameplay:** Retro grid-based movement with a custom-designed `tileMap`.
*   **Dynamic Ghost AI:** Four distinct ghosts (`Blinky`, `Pinky`, `Inky`, and `Clyde`) that move dynamically throughout the maze.
*   **Smooth Controls:** Responsive directional movement coupled with animated Pac-Man sprites that change direction based on user input.
*   **Game State Management:** Implements live tracking (3 lives), scoring system (+10 per food pellet), map resetting upon level completion, and a Game Over state.
*   **Lightweight Performance:** Built efficiently using native Java standard libraries (`javax.swing.Timer` and `Graphics`).

---

## 🛠️ Built With

*   **Language:** Java 8+
*   **Framework/GUI:** Java Swing & AWT

---

## 🎮 How to Play

1.  **Controls:** Use the **Arrow Keys** (`Up`, `Down`, `Left`, `Right`) to navigate Pac-Man through the maze.
2.  **Objective:** Eat all the white food pellets on the map while avoiding the ghosts.
3.  **Game Over:** If a ghost touches you, you lose a life. Lose all 3 lives, and it's Game Over.
4.  **Restart:** Press **any arrow key** after a Game Over to restart the game instantly.
